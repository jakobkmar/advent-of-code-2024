# Advent of Code 2024

My Advent of Code 2024 repo! This repository contains solutions and sometimes explanations for the Advent of Code 2024 challenges.

## Running the Code

Place your input alongside the day folders.

**Python**:
```sh
python day01.py
```

**Typst**:
```sh
typst compile day01.typ
```
